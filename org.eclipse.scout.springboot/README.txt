Running the example in Eclipse IDE:
===================================
Ensure Lombok is added to your JRE (https://projectlombok.org/)

Run 'org.eclipse.scout.springboot.Application' as normal Java program

Running the example in console:
===============================
mvn spring-boot:run

Open Application on http://localhost:8080/

By default, this application requires user authentication. The credentials are 'eclipse/scout', 'alice/test', 'bob/test'.
